# 프로퍼티
프로퍼티는 클래스에 속한 `var`, `val`이다.

```kotlin
class Cup{
  var percentFull = 0
  val max = 100
  fun add(increase: Int): Int{
    percentFull += increase
    if(percentFull > max)
      percentFull = max
    return percentFull
  }
}

fun main() {
  val cup = Cup()
  cup.add(50)
  println(cup.percentFull)
  cup.add(70)
  println(cup.percentFull)
}

```
