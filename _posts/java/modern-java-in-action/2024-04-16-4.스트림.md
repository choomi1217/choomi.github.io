### 스트림 예시
```java
public class Example1 {

  private static final List<Dish> dishes = List.of(
    new Dish("피자", false, 1000)
    , new Dish("햄버거", false, 500)
    , new Dish("샐러드", true, 400)
    , new Dish("콜라", false, 150));

  public static void main(String[] args) {
    dishes.stream().filter(dish -> dish.getCalories() < 400)
      .sorted((d1, d2) -> Integer.compare(d1.getCalories(), d2.getCalories()))
      .map(Dish::getName)
      .forEach(System.out::println);
  }
}
```
